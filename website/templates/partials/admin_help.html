Enterprise Leadership

"Enterprise Leadership" page in the top nav. Hero on top, Scope and Impact below it, three function blocks underneath.

  • Hero heading and lead paragraph at the top of the page. Enterprise Overview → Hero heading / Hero lead paragraph
  • "Scope" and "Impact" blocks directly below the hero. Enterprise Overview → Scope / Impact
  • "What I lead" heading above the three function blocks. Enterprise Overview → Function section heading
  • The three function blocks (Compensation, HRIS, Payroll) — title, summary, and three body blocks (Responsibilities, Systems Led, Role in the Organization). Enterprise Functions. Reorder with Position.
  • "Systems Led" and "Role in the Organization" eyebrow labels — small uppercase tags above those two blocks. Editable per function (defaults: "PLATFORMS AND TOOLS" / "ROLE AND ACCOUNTABILITY"). Enterprise Functions → "Systems Led" section label / "Role in the Organization" section label
  • "How the system works together" closing block at the bottom. Enterprise Overview → System integration heading / body

The "Scope", "Impact", and "Responsibilities" eyebrow labels are still fixed in code — they label the structure of the page, not the content inside it.

Editing conventions

  • Headings live in their own short text fields, never inside the rich-text editor. The editor strips heading tags by design.
  • Site Configuration, Enterprise Overview, Innovation Overview, and Connect Page are single-record screens. Edit the existing one — never create a second.
  • Your edits are the source of truth. Deploys never overwrite admin edits. (If a fixture-driven rebuild is ever needed, it now requires an explicit one-shot command.)
  • Save and refresh — admin edits appear on the public site immediately. No cache to clear.
